# Expo HAS CHANGED

Read the exact versioned docs at https://docs.expo.dev/versions/v57.0.0/ before writing any code.

For local async key-value storage, `expo-sqlite/kv-store` is the single source of truth. Do not install `@react-native-async-storage/async-storage`, do not use `expo-sqlite/localStorage/install`, and do not create manual SQLite tables for favorite or map pin key-value data.
