# Expo HAS CHANGED

Read the exact versioned docs at https://docs.expo.dev/versions/v57.0.0/ before writing any code.

For local key-value storage, `react-native-mmkv` is the single source of truth. Do not use `@react-native-async-storage/async-storage`, `expo-sqlite/kv-store`, `expo-sqlite/localStorage/install`, or manual SQLite tables for favorite or map pin key-value data.
